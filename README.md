### Agvents - The Advance of Events ™©

#### Projeto:
Tivemos a ideia desse projeto tendo como base a maneira com que o agendamento de eventos é realizada hoje, de uma forma normalmente lenta e não muito agil, cujo a partir de tal ponto tivemos a epifania da criação da Agvents, cujo pode ser lida como Advance pois ela é a Agile Events a forma ágil, prática e avançada de se realizar tais agendamentos de eventos desta maneira.

#### Metodologia:
Programação frontend e backend, Fullstack, Desenvolvimento web e Mobile, Agendamento de eventos e Desenvolvimento.

#### Objetivo:
Desenvolvimento um sistema rápido e fácil de gestão do agendamento de eventos por meio de uma aplicação web e um aplicativo de fácil acesso, possuindo uma collection em um cluster do mongodb atlas, uma api graphql em node, aplicação erb em react e um aplicativo em flutter.

#### Tecnologias utilizadas:
React.js, Node.js, Vscode, Javascript, Flutter, GraphQl, Collection, Mongodb Atlas, Github
